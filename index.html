01 — Research

What I work on

Full research statement

I study how AI agents behave on real software systems: what context they read, which actions they choose, and what evidence they leave behind.

01 AI for SRE SREGym with UIUC and Tianyin Xu's group

An AI-native benchmark for Site Reliability Engineering agents: real incidents, real systems, machine-checkable evidence.

 02 Agent security SHIFT aligned-backdoor audit with Chowdhury Rakin Haider

Detecting triggers that steer an agent's choice between valid options toward an attacker's target. Submitted to TACL 2026.

 03 Coding agents ContextLedger, ctxhelm, PatchSmith, and history-aware coding

Local-first tooling that makes agent context, repair loops, and repository history measurable instead of anecdotal.

 04 Software evolution RefactoringMiner MCP work with Nikolaos Tsantalis and JetBrains

Connecting refactoring analysis with coding agents through an MCP server, in active collaboration with the maintainers.

Research & publications

All publications

An Empirical Study on Remote Code Execution in Machine Learning Model Hosting Ecosystems

ML model hosting security · Large-scale empirical study

Large-scale study of roughly 45,000 repositories across major ML model hosting ecosystems, measuring unsafe deserialization, eval injection, malware signals, and recurring developer security misconceptions.

The Choice Can Be the Attack: Auditing Aligned Backdoors in LLM Agents

Submitted to TACL 2026 · SHIFT audit

SHIFT checks whether a trigger changes which valid option an agent chooses, and whether that movement favors the attacker's target. With Chowdhury Rakin Haider.

VeriSchema: Multi-Agent Framework for Generating Relational DB Schema & ERD

Multi-agent verification · Schema generation

Multi-agent framework for generating relational database schemas and ER diagrams from requirements, with auto-repair and formal verification.
