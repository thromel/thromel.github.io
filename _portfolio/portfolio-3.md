An enhanced version of the "Show, Attend, and Tell" image captioning model that incorporates BERT for improved language understanding:

* Integration of BERT for enhanced language context
* Improved attention mechanisms for better image feature extraction
* Faster training convergence through transfer learning
* Enhanced caption quality and relevance

## Key Achievements
* Improved caption quality through BERT's language understanding
* Reduced training time while maintaining accuracy
* Better context awareness in generated captions
* Successful combination of computer vision and NLP techniques

* PyTorch implementation
* BERT integration for language modeling
* CNN-based image feature extraction
* Attention mechanism for feature focusing
