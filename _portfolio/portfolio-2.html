Eventfly is an end-to-end event management system built with modern microservices architecture:

Key Features

  • Comprehensive event organization and participation platform
  • Microservices-based architecture for scalability
  • Real-time newsfeed for event updates
  • Integrated payment system
  • Secure authentication and authorization

Technical Stack

  • TypeScript and Express.js for backend services
  • Next.js for the frontend
  • Docker and Kubernetes for containerization and orchestration
  • NATS for message queuing and service communication
  • MongoDB for data persistence

Architecture Highlights

  • Event-driven architecture using NATS
  • Containerized microservices with Docker
  • Kubernetes for service orchestration
  • RESTful APIs for service communication
  • Responsive and modern UI with Next.js
