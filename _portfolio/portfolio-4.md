**Objective**: Enable secure, decentralized sharing of patient health records while maintaining HIPAA compliance.

## Tech Stack
### Blockchain
* Ethereum
* Solidity
* Web3.js
* IPFS

### Backend & Frontend
* Node.js
* React.js
* Express.js
* MongoDB (for off-chain data)

### Security
* AES-256 encryption
* RSA public-key cryptography
* SHA-3 hashing
* JWT authentication

## System Architecture
* **Smart Contracts**:
  * Role-based access control
  * Data access logging
  * Consent management
  * Key management
  * Record indexing

* **Storage Layer**:
  * IPFS for distributed storage
  * Encrypted medical records
  * Cryptographic proofs
  * Access logs
  * Audit trails

* **Access Control**:
