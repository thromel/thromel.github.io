An innovative blockchain-based framework for secure storage and sharing of patient health records, developed as an undergraduate thesis project:

## System Architecture
* Hybrid encryption using AES (symmetric) and RSA (asymmetric)
* Smart contracts on Ethereum for access control
* Cryptographic hashing for data integrity
* Decentralized key management system
* Structured medical data sharing protocol

## Key Features
* Secure storage of patient health records
* Granular access control through smart contracts
* Tamper-proof audit trail
* Near real-time encryption/decryption
* Selective data sharing capabilities
* HIPAA compliance considerations

## Technical Implementation
* Ethereum blockchain platform
* Solidity smart contracts
* Web3.js for blockchain interaction
* AES-256 for data encryption
* RSA for key exchange
* IPFS for distributed storage

## Impact
* Enhanced patient data security
* Improved healthcare data interoperability
* Efficient medical history sharing
* Reduced risk of data breaches
* Transparent audit trails
